FLIXGO – Best Place for Recommend Movies

Our project belongs to the creation of a Web Application.A user search a movie name he already seen. Our goal is to suggest movies based on the user input for achieve this goal we use some Dataset that contain informations like Cast, Genre , Description , Director etc .. to predict movies using Machine Learning approach. We also provide Ratings and other informations of suggested movies from various sites like Rotten Tomatoes ,imdb etc..

Our project aims to provide a content based movie recommendation system.Users can find similar movies instantly.
